# ⛽ Gas Station Shift Manager (Ταμείο Βάρδιας Βενζινάδικου)

Μια ελαφριά, web-based εφαρμογή για την εύκολη διαχείριση και καταγραφή των καθημερινών οικονομικών στοιχείων μιας βάρδιας σε πρατήριο υγρών καυσίμων.

Ο **Shift Manager** επιτρέπει στους υπαλλήλους να καταγράφουν πωλήσεις, πιστώσεις και έξοδα σε πραγματικό χρόνο, δημιουργώντας μια καθαρή αναφορά έτοιμη για εκτύπωση.

---

## 🚀 Χαρακτηριστικά

* **Δυναμική Προσθήκη Δεδομένων:** Προσθέστε όσες γραμμές χρειάζεστε για λάδια, αξεσουάρ, έξοδα και εισπράξεις.
* **Διαχείριση Πιστώσεων:** Ειδική φόρμα για καταγραφή Πελάτη, Πινακίδας και Ποσού.
* **Αυτόματος Υπολογισμός:** Τα σύνολα ενημερώνονται αυτόματα με κάθε πληκτρολόγηση (Real-time Reporting).
* **Τοπική Αποθήκευση (Persistence):** Τα δεδομένα δεν χάνονται αν κλείσετε ή ανανεώσετε τη σελίδα (χρήση `localStorage`).
* **Λειτουργία Εκτύπωσης:** Βελτιστοποιημένη εμφάνιση για εκτύπωση της αναφοράς σε χαρτί ή PDF.
* **Προστασία Δεδομένων:** Προειδοποίηση πριν την έξοδο από τη σελίδα για την αποφυγή απώλειας μη αποθηκευμένων στοιχείων.

---

## 🛠 Τεχνολογίες

Η εφαρμογή είναι κατασκευασμένη με **"Vanilla"** τεχνολογίες, χωρίς την ανάγκη εξωτερικών βιβλιοθηκών ή server:

* **HTML5:** Δομή και φόρμες εισαγωγής.
* **CSS3:** Layout και styling (συμπεριλαμβανομένων print media queries).
* **JavaScript (ES6):** Λογική υπολογισμών, διαχείριση DOM και αποθήκευση δεδομένων.

## 📖 Τρόπος Χρήσης

1.  **Στοιχεία Βάρδιας:** Εισάγετε τα ονόματα των υπαλλήλων και την ημερομηνία.
2.  **Καταχώρηση Πωλήσεων:** Πατήστε τα πράσινα κουμπιά "➕" για να προσθέσετε νέα είδη (π.χ. Λάδια, Αξεσουάρ).
3.  **Πιστώσεις:** Συμπληρώστε το όνομα του πελάτη και την πινακίδα του οχήματος για την παρακολούθηση των οφειλών.
4.  **Έξοδα/Εισπράξεις:** Καταγράψτε τυχόν πληρωμές από το ταμείο ή εισπράξεις παλαιών οφειλών.
5.  **Εκτύπωση:** Μόλις ολοκληρώσετε, πατήστε "Εκτύπωση" για να λάβετε το έγγραφο της αναφοράς.
6.  **Νέα Βάρδια:** Πατήστε "Καθαρισμός" για να διαγράψετε όλα τα τρέχοντα δεδομένα και να ξεκινήσετε από το μηδέν.
